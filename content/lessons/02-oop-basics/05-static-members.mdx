# Static Members in C#

Static members belong to the class itself rather than individual instances. They're shared across all objects of that class.

## Static vs Instance Members

<CodeEditor 
  initialCode={`using System;

class Counter
{
    // Static field - shared across all instances
    public static int TotalCount = 0;
    
    // Instance field - each object has its own
    public int InstanceId;
    
    public Counter()
    {
        TotalCount++;           // Increment shared counter
        InstanceId = TotalCount; // Assign unique ID
    }
}

class Program
{
    static void Main()
    {
        Console.WriteLine($"Before: TotalCount = {Counter.TotalCount}");
        
        Counter c1 = new Counter();
        Console.WriteLine($"Created c1: ID = {c1.InstanceId}, Total = {Counter.TotalCount}");
        
        Counter c2 = new Counter();
        Console.WriteLine($"Created c2: ID = {c2.InstanceId}, Total = {Counter.TotalCount}");
        
        Counter c3 = new Counter();
        Console.WriteLine($"Created c3: ID = {c3.InstanceId}, Total = {Counter.TotalCount}");
        
        // Static accessed via class name
        Console.WriteLine($"\\nFinal count: {Counter.TotalCount}");
    }
}`}
/>

## Static Fields

<CodeEditor 
  initialCode={`using System;

class BankAccount
{
    // Static fields - shared data
    private static decimal interestRate = 0.05m;  // 5%
    private static int accountCount = 0;
    
    // Instance fields - per-account data
    public string AccountNumber { get; }
    public decimal Balance { get; private set; }
    
    public BankAccount(decimal initialBalance)
    {
        accountCount++;
        AccountNumber = $"ACC-{accountCount:D4}";
        Balance = initialBalance;
    }
    
    // Static method to access/modify static field
    public static void SetInterestRate(decimal rate)
    {
        if (rate >= 0 && rate <= 0.20m)
        {
            interestRate = rate;
            Console.WriteLine($"Interest rate set to {rate:P1}");
        }
    }
    
    public static decimal GetInterestRate() => interestRate;
    public static int GetAccountCount() => accountCount;
    
    // Instance method using static field
    public void AddInterest()
    {
        decimal interest = Balance * interestRate;
        Balance += interest;
        Console.WriteLine($"{AccountNumber}: Added ${'{'}interest:F2{'}'} interest");
    }
}

class Program
{
    static void Main()
    {
        Console.WriteLine($"Interest rate: {BankAccount.GetInterestRate():P1}");
        
        var acc1 = new BankAccount(1000);
        var acc2 = new BankAccount(2000);
        
        Console.WriteLine($"\\nAccounts created: {BankAccount.GetAccountCount()}");
        
        acc1.AddInterest();
        acc2.AddInterest();
        
        // Change rate for ALL accounts
        BankAccount.SetInterestRate(0.07m);
        
        acc1.AddInterest();
        acc2.AddInterest();
    }
}`}
/>

## Static Properties

<CodeEditor 
  initialCode={`using System;

class Application
{
    // Private static backing field
    private static string version = "1.0.0";
    
    // Static auto-property
    public static string Name { get; set; } = "My App";
    
    // Static property with validation
    public static string Version
    {
        get => version;
        set
        {
            if (!string.IsNullOrWhiteSpace(value))
            {
                version = value;
                Console.WriteLine($"Version updated to {value}");
            }
        }
    }
    
    // Read-only static property
    public static DateTime StartTime { get; } = DateTime.Now;
    
    // Computed static property
    public static TimeSpan Uptime => DateTime.Now - StartTime;
}

class Program
{
    static void Main()
    {
        Console.WriteLine($"App Name: {Application.Name}");
        Console.WriteLine($"Version: {Application.Version}");
        Console.WriteLine($"Started: {Application.StartTime}");
        
        Application.Name = "Super App";
        Application.Version = "2.0.0";
        
        Console.WriteLine($"\\nUpdated Name: {Application.Name}");
        
        // Simulate some work
        System.Threading.Thread.Sleep(1000);
        
        Console.WriteLine($"Uptime: {Application.Uptime.TotalSeconds:F1} seconds");
    }
}`}

## Static Methods

<CodeEditor 
  initialCode={`using System;

class MathHelper
{
    // Static methods - utility functions
    public static int Max(int a, int b)
    {
        return a > b ? a : b;
    }
    
    public static int Min(int a, int b)
    {
        return a < b ? a : b;
    }
    
    public static double Average(params double[] numbers)
    {
        if (numbers.Length == 0) return 0;
        
        double sum = 0;
        foreach (var n in numbers)
        {
            sum += n;
        }
        return sum / numbers.Length;
    }
    
    public static bool IsPrime(int number)
    {
        if (number < 2) return false;
        if (number == 2) return true;
        if (number % 2 == 0) return false;
        
        for (int i = 3; i <= Math.Sqrt(number); i += 2)
        {
            if (number % i == 0) return false;
        }
        return true;
    }
    
    public static int Factorial(int n)
    {
        if (n < 0) throw new ArgumentException("Must be non-negative");
        if (n <= 1) return 1;
        return n * Factorial(n - 1);
    }
}

class Program
{
    static void Main()
    {
        // Call static methods without creating instance
        Console.WriteLine($"Max(5, 10) = {MathHelper.Max(5, 10)}");
        Console.WriteLine($"Min(5, 10) = {MathHelper.Min(5, 10)}");
        Console.WriteLine($"Average(1,2,3,4,5) = {MathHelper.Average(1, 2, 3, 4, 5)}");
        
        Console.WriteLine("\\nPrime numbers from 1-20:");
        for (int i = 1; i <= 20; i++)
        {
            if (MathHelper.IsPrime(i))
            {
                Console.Write($"{i} ");
            }
        }
        
        Console.WriteLine($"\\n\\nFactorial(5) = {MathHelper.Factorial(5)}");
        Console.WriteLine($"Factorial(10) = {MathHelper.Factorial(10)}");
    }
}`}
/>

## Static Constructors

<CodeEditor 
  initialCode={`using System;
using System.Collections.Generic;

class Database
{
    // Static fields to initialize
    public static string ConnectionString { get; private set; }
    public static bool IsConnected { get; private set; }
    private static List<string> tables;
    
    // Static constructor - runs once before first access
    static Database()
    {
        Console.WriteLine("Static constructor running...");
        
        // Initialize static members
        ConnectionString = "Server=localhost;Database=test";
        tables = new List<string> { "Users", "Products", "Orders" };
        IsConnected = true;
        
        Console.WriteLine("Database initialized!\\n");
    }
    
    public static void ListTables()
    {
        Console.WriteLine("Available tables:");
        foreach (var table in tables)
        {
            Console.WriteLine($"  - {table}");
        }
    }
    
    public static void AddTable(string name)
    {
        tables.Add(name);
        Console.WriteLine($"Added table: {name}");
    }
}

class Program
{
    static void Main()
    {
        Console.WriteLine("Program starting...");
        
        // First access triggers static constructor
        Console.WriteLine($"Connected: {Database.IsConnected}");
        Console.WriteLine($"Connection: {Database.ConnectionString}\\n");
        
        Database.ListTables();
        
        Database.AddTable("Inventory");
        Console.WriteLine();
        Database.ListTables();
    }

## Static Classes

Classes that can only contain static members:

<CodeEditor 
  initialCode={`using System;

// Static class - cannot be instantiated
static class StringUtils
{
    public static string Reverse(string input)
    {
        char[] chars = input.ToCharArray();
        Array.Reverse(chars);
        return new string(chars);
    }
    
    public static string Capitalize(string input)
    {
        if (string.IsNullOrEmpty(input)) return input;
        return char.ToUpper(input[0]) + input.Substring(1).ToLower();
    }
    
    public static int WordCount(string input)
    {
        if (string.IsNullOrWhiteSpace(input)) return 0;
        return input.Split(new[] { ' ', '\\t', '\\n' }, 
            StringSplitOptions.RemoveEmptyEntries).Length;
    }
    
    public static bool IsPalindrome(string input)
    {
        string cleaned = input.ToLower().Replace(" ", "");
        string reversed = Reverse(cleaned);
        return cleaned == reversed;
    }
}

static class NumberUtils
{
    public static bool IsEven(int n) => n % 2 == 0;
    public static bool IsOdd(int n) => n % 2 != 0;
    public static bool IsPositive(int n) => n > 0;
    public static bool IsNegative(int n) => n < 0;
}

class Program
{
    static void Main()
    {
        // Cannot do: var utils = new StringUtils(); // Compile error!
        
        string text = "Hello World";
        Console.WriteLine($"Original: {text}");
        Console.WriteLine($"Reversed: {StringUtils.Reverse(text)}");
        Console.WriteLine($"Capitalized: {StringUtils.Capitalize(text)}");
        Console.WriteLine($"Word count: {StringUtils.WordCount(text)}");
        
        string palindrome = "A man a plan a canal Panama";
        Console.WriteLine($"\\n'{palindrome}'");
        Console.WriteLine($"Is palindrome: {StringUtils.IsPalindrome(palindrome)}");
        
        Console.WriteLine("\\nNumber checks:");
        Console.WriteLine($"4 is even: {NumberUtils.IsEven(4)}");
        Console.WriteLine($"7 is odd: {NumberUtils.IsOdd(7)}");
    }
}`}

## Singleton Pattern

Using static to ensure only one instance:

<CodeEditor 
  initialCode={`using System;
using System.Collections.Generic;

class GameManager
{
    // Static instance holder
    private static GameManager instance;
    
    // Instance properties
    public int Score { get; private set; }
    public int Level { get; private set; }
    public List<string> Achievements { get; } = new List<string>();
    
    // Private constructor prevents external instantiation
    private GameManager()
    {
        Score = 0;
        Level = 1;
        Console.WriteLine("GameManager created!");
    }
    
    // Static property to get the singleton instance
    public static GameManager Instance
    {
        get
        {
            if (instance == null)
            {
                instance = new GameManager();
            }
            return instance;
        }
    }
    
    // Instance methods
    public void AddScore(int points)
    {
        Score += points;
        Console.WriteLine($"Score: {Score} (+{points})");
        
        if (Score >= Level * 100)
        {
            LevelUp();
        }
    }
    
    private void LevelUp()
    {
        Level++;
        Console.WriteLine($"*** LEVEL UP! Now at level {Level} ***");
        Achievements.Add($"Reached Level {Level}");
    }
    
    public void ShowStats()
    {
        Console.WriteLine($"\\nGame Stats:");
        Console.WriteLine($"  Score: {Score}");
        Console.WriteLine($"  Level: {Level}");
        Console.WriteLine($"  Achievements: {Achievements.Count}");
    }
}

class Program
{
    static void Main()
    {
        // Access via singleton property
        GameManager.Instance.AddScore(50);
        GameManager.Instance.AddScore(75);
        GameManager.Instance.AddScore(100);
        
        // Another reference to same instance
        var game = GameManager.Instance;
        game.AddScore(200);
        
        GameManager.Instance.ShowStats();
        
        Console.WriteLine($"\\nSame instance: {game == GameManager.Instance}");
    }
}`}
/>

## Extension Methods

Static methods that extend existing types:

<CodeEditor 
  initialCode={`using System;

// Extension methods must be in a static class
static class StringExtensions
{
    // 'this' keyword makes it an extension method
    public static string Repeat(this string str, int count)
    {
        return string.Concat(System.Linq.Enumerable.Repeat(str, count));
    }
    
    public static bool IsNullOrEmpty(this string str)
    {
        return string.IsNullOrEmpty(str);
    }
    
    public static string TruncateWithEllipsis(this string str, int maxLength)
    {
        if (str.Length <= maxLength) return str;
        return str.Substring(0, maxLength - 3) + "...";
    }
}

static class IntExtensions
{
    public static bool IsBetween(this int num, int min, int max)
    {
        return num >= min && num <= max;
    }
    
    public static string ToOrdinal(this int num)
    {
        string suffix = (num % 100) switch
        {
            11 or 12 or 13 => "th",
            _ => (num % 10) switch
            {
                1 => "st",
                2 => "nd",
                3 => "rd",
                _ => "th"
            }
        };
        return $"{num}{suffix}";
    }
}

class Program
{
    static void Main()
    {
        // Call extension method like instance method
        string hello = "Hello! ";
        Console.WriteLine(hello.Repeat(3));
        
        string longText = "This is a very long string that needs truncation";
        Console.WriteLine(longText.TruncateWithEllipsis(20));
        
        string empty = "";
        Console.WriteLine($"Empty string is null or empty: {empty.IsNullOrEmpty()}");
        
        Console.WriteLine("\\nNumber extensions:");
        int num = 42;
        Console.WriteLine($"{num} is between 1 and 50: {num.IsBetween(1, 50)}");
        
        for (int i = 1; i <= 5; i++)
        {
            Console.WriteLine($"{i} -> {i.ToOrdinal()}");
        }
    }
}`}

## Complete Example: Logger System

<CodeEditor 
  initialCode={`using System;
using System.Collections.Generic;

static class Logger
{
    // Static fields
    private static List<string> logs = new List<string>();
    private static bool isEnabled = true;
    private static LogLevel minLevel = LogLevel.Info;
    
    // Enum for log levels
    public enum LogLevel { Debug = 0, Info = 1, Warning = 2, Error = 3 }
    
    // Static properties
    public static bool IsEnabled
    {
        get => isEnabled;
        set => isEnabled = value;
    }
    
    public static LogLevel MinimumLevel
    {
        get => minLevel;
        set => minLevel = value;
    }
    
    public static int LogCount => logs.Count;
    
    // Static methods
    public static void Log(LogLevel level, string message)
    {
        if (!isEnabled || level < minLevel) return;
        
        string entry = $"[{DateTime.Now:HH:mm:ss}] [{level}] {message}";
        logs.Add(entry);
        
        // Color output based on level
        ConsoleColor color = level switch
        {
            LogLevel.Debug => ConsoleColor.Gray,
            LogLevel.Info => ConsoleColor.White,
            LogLevel.Warning => ConsoleColor.Yellow,
            LogLevel.Error => ConsoleColor.Red,
            _ => ConsoleColor.White
        };
        
        Console.ForegroundColor = color;
        Console.WriteLine(entry);
        Console.ResetColor();
    }
    
    // Convenience methods
    public static void Debug(string msg) => Log(LogLevel.Debug, msg);
    public static void Info(string msg) => Log(LogLevel.Info, msg);
    public static void Warning(string msg) => Log(LogLevel.Warning, msg);
    public static void Error(string msg) => Log(LogLevel.Error, msg);
    
    public static void ShowAllLogs()
    {
        Console.WriteLine($"\\n=== All Logs ({logs.Count}) ===");
        foreach (var log in logs)
        {
            Console.WriteLine(log);
        }
    }
    
    public static void Clear()
    {
        logs.Clear();
        Console.WriteLine("Logs cleared.");
    }
}

class Program
{
    static void Main()
    {
        Logger.Info("Application started");
        Logger.Debug("Debug mode enabled");  // Won't show by default
        Logger.Warning("Low memory warning");
        Logger.Error("Failed to connect to database");
        
        Console.WriteLine($"\\nLog count: {Logger.LogCount}");
        
        // Enable debug logging
        Logger.MinimumLevel = Logger.LogLevel.Debug;
        Logger.Debug("Now debug messages show up!");
        
        // Disable logging
        Logger.IsEnabled = false;
        Logger.Info("This won't appear");
        
        Logger.IsEnabled = true;
        Logger.Info("Logging re-enabled");
        
        Logger.ShowAllLogs();
    }
}`}
/>

## Key Takeaways

- **Static members** belong to the class, not instances
- **Static fields** are shared across all objects
- **Static methods** can be called without creating an instance
- **Static constructors** initialize static members once
- **Static classes** contain only static members and cannot be instantiated
- **Extension methods** add functionality to existing types

## When to Use Static

| Use Static For | Use Instance For |
|----------------|------------------|
| Utility/helper methods | Object-specific data |
| Constants and configuration | State that varies per object |
| Factory methods | Behavior tied to object state |
| Singleton patterns | Polymorphic behavior |
| Extension methods | Data encapsulation |

## Best Practices

1. **Use static sparingly** - overuse makes testing difficult
2. **Consider thread safety** for static fields
3. **Prefer extension methods** over utility classes when possible
4. **Use static readonly** for constants that need runtime initialization
5. **Avoid mutable static state** when possible
